## Virtual Zoo 🦁

- A Java console-based application that simulates a simple zoo management system using Object-Oriented Programming (OOP) concepts and CRUD operations.

- This project demonstrates how real-world entities (animals in a zoo) can be modeled using Java classes and managed through a menu-driven command line interface.

---

## 🚀 Features

- Add animals to the zoo
- View all animals
- Update animal information
- Delete animals from the zoo
- Menu-driven console interface
- Demonstrates Java OOP principles

---

## 🛠 Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Java Collections
- Command Line Interface (CLI)

---

## 📚 OOP Concepts Implemented

This project demonstrates the following OOP principles:

- Encapsulation – grouping data and methods inside classes

- Inheritance – creating specialized animal types

- Abstraction – defining common animal behaviors

- Modularity – separating logic across classes

## ▶️ How to Run the Project
### 1️⃣ Clone the repository

git clone https://github.com/dobhalsaurish/Virtual-Zoo.git

### 2️⃣ Navigate to the project folder

cd Virtual-Zoo

### 3️⃣ Compile the Java files

javac *.java

### 4️⃣ Run the program

java Main
